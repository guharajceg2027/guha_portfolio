# Guharaj Muralitharan | Portfolio Website

This is the source code for my personal portfolio website. It highlights my academic achievements, technical skills, certifications, and projects. The site is designed to be clean, responsive, and accessible across devices.

---

## Overview

The website provides visitors with insight into who I am as an IT student, problem solver, and future civil services aspirant. Key sections include:

- About Me
- Skills
- Projects
- Certifications
- Contact Information

It also features a resume download option and a certificate preview for the Arvind Mehta Memorial Prize awarded by AACEG, Anna University.

---

## Features

- Well-structured HTML with a light color theme
- Responsive layout using Bootstrap 5
- Use of Bootstrap Icons for visual clarity
- Resume download via Google Drive or local assets
- Organized cards for project and certification displays
- Clear and accessible navigation bar

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Basic JavaScript
